# Blinko Simple Styles

This repo contains a plugin for Blinko for changing web app's styles and CSS variables.

## Installation

1. Plugin -> Marketplace
2. Click Install from GitHub
3. Enter `https://github.com/madcativ/blinko-simple-styles`
4. Click Install

## Usage

1. Plugin -> Installed
2. Find blinko-note-editor-colors
3. Click the cog icon to open settings
4. Edit CSS Variables and Styles as needed

### Example CSS Variables (JSON)

```json
{
	"primary": "#fff",
	"secondary": "#000"
}
```
#### Note:

Do not include the `--` prefix when defining CSS variables.

### Example Styles (CSS)

```css
body {
	color : #fff;
}
```

#### Note:

Use valid CSS syntax for styles.
